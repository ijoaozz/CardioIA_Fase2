# 📘 Dicionário de Dados -- Dataset Doença Cardíaca

Este documento descreve o significado de cada variável presente no
dataset utilizado no projeto CardioIA_Fase1.

------------------------------------------------------------------------

## 🔢 Variáveis Numéricas

-   **idade**: Idade do paciente em anos.
-   **pressao_repouso**: Pressão arterial em repouso (mm Hg).
-   **colesterol**: Colesterol sérico em mg/dl.
-   **freq_cardiaca_max**: Frequência cardíaca máxima atingida.
-   **pico_antigo**: Depressão do segmento ST induzida por exercício.
-   **vasos_principais**: Número de vasos principais coloridos por
    fluoroscopia (0--3).

------------------------------------------------------------------------

## 🧍 Variáveis Categóricas Codificadas

### sexo

-   0 = Feminino\
-   1 = Masculino

### tipo_dor_peito

-   0 = Angina típica\
-   1 = Angina atípica\
-   2 = Dor não anginosa\
-   3 = Assintomático

### glicose_jejum

-   0 = ≤ 120 mg/dl\
-   1 = \> 120 mg/dl

### eletrocardiograma

-   0 = Normal\
-   1 = Anormalidade da onda ST-T\
-   2 = Hipertrofia ventricular esquerda

### angina_exercicio

-   0 = Não\
-   1 = Sim

### inclinacao_st

-   0 = Ascendente\
-   1 = Plano\
-   2 = Descendente

### talassemia

-   0 = Normal\
-   1 = Defeito fixo\
-   2 = Defeito reversível

### doenca_cardiaca (Variável Alvo)

-   0 = Não possui doença cardíaca\
-   1 = Possui doença cardíaca

------------------------------------------------------------------------

projeto.
